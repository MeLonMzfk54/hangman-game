# Игра висилица

Чтобы запустить локально:
1) Скачайте исходный код с репозитория https://github.com/MeLonMzfk54/hangman-game
2) Откройте консоль, находясь в директории проекта, и напишите "npm i", чтобы установить необходимые пакеты
3) В консоли, находясь в этой же директории, напишите npm run dev

## Проект написан на Vue3 + TypeScript
